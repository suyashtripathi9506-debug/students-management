# 🎓 Student Management System (Python)

## 📌 Overview

This is a simple **Student Management System** built using Python.
It allows users to manage student records directly from the terminal.

---

## 🚀 Features

* 📋 View all students
* ➕ Add new students
* 🔍 Search for a student
* ❌ Remove a student
* 🔁 Continuous menu-driven system

---

## 🛠️ Technologies Used

* Python 3
* Basic Data Structures (List)
* Functions & Loops

---

## 📂 Project Structure

```
student_management.py   # Main Python file
README.md               # Project documentation
```

---

## ▶️ How to Run

1. Install Python (if not installed)
2. Download or clone this repository
3. Open terminal / command prompt
4. Run the file:

```
python student_management.py
```

---

## 📖 Usage

* Enter a number (1–5) from the menu:

  * **1** → Show students
  * **2** → Add student
  * **3** → Find student
  * **4** → Remove student
  * **5** → Exit

---

## 💡 Example

```
Enter number between 1 to 5: 2
Enter student name: rahul
rahul added successfully
```
